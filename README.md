# there-is-no-clicker-here

Un clicker narratif étrange.

Tu cliques.
Le jeu répond.
Puis il commence à te répondre un peu trop.

## Pourquoi y jouer

- Progression en paliers avec changements visuels et sonores
- Ambiance absurde, sarcastique, imprévisible
- Secrets cachés à débloquer
- Sauvegarde automatique de ta progression

## Lancer le jeu (local)

Le jeu sera déployé prochainement.
Tu pourras y jouer [ici]().

## Comment jouer

- Clique n'importe où dans la fenêtre.
- Avance jusqu'aux paliers pour débloquer des effets et des découvertes.
- Ouvre le panneau des découvertes via l'étoile `★` (quand elle apparaît).
- Continue de cliquer: le jeu garde des surprises pour la suite.

## Raccourcis utiles

- `A` : ouvre/ferme le panneau des découvertes (une fois débloqué)
- `Esc` : ferme le panneau

## Sauvegarde

La progression est sauvegardée automatiquement dans le navigateur.
Attention, les sauvegardes sont perdues si tu nettoies les cookies du navigateur. 

*Et ça serait bête de tout perdre, non ?*

## Réinitialiser la progression

Tape `reset` au clavier pendant la partie, puis confirme.
